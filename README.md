# collab-task

Shared repository for direct collaborator workflow.

## How we work

1. **Owner** invited collaborators with **Write** access (push + open PRs).
2. **Collaborators** create a branch, commit work, push, then open a **Pull Request** into `main`.
3. **Owner** reviews the PR and **Merge** when ready.
